# Практика
Репозиторий содержит учебный проект, выполненный в рамках производственной практики.

## Описание
В ходе работы выполнялось сопровождение и обслуживание готового программного проекта: анализ структуры, настройка окружения, запуск приложения, проверка качества кода, улучшение защиты, обновление документации и работа с Git.

## Используемые технологии
- C# / ASP.NET Core Web API
- Python / Django
- PostgreSQL
- HTML, CSS, JavaScript
- Git / GitHub

## Основные выполненные работы
- Клонирование и анализ структуры проекта
- Настройка окружения и запуск backend/frontend
- Анализ качества кода
- Добавление логирования
- Улучшение базовой защиты
- Обновление документации
- Добавление CHANGELOG
- Работа с ветками и коммитами
